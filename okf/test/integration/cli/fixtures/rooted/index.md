# Rooted

A bundle whose tagged concepts live at the root *and* in an area — the shape
that makes `tags --by area` print both of its labels: `(root)` bare, and a
nested area with its trailing slash.

* [Charter](charter.md) - a tagged concept that lives at the bundle root.

# Areas

* [Services](services/) - one tagged concept, one directory deep.
