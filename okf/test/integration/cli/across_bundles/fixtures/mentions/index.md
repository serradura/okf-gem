# Mentions

A bundle whose prose carries literal @-handles. Every other fixture is @-free,
so nothing could prove that `search -e '\@handle'` *finds* the text: a leading
@arg is read as a registry ref, and the escape hatch that gets past that has to
land on something to be worth documenting.

* [Escalation](escalation.md) - who to page when checkout errors spike.
* [Ownership](ownership.md) - which team owns the payments service.
